# 🚖 Cab Fare Prediction System

A Machine Learning powered web application that predicts taxi fare based on passenger count, distance, and time features. Built using Flask and trained using Linear Regression.

---

## 📌 Project Overview

This project predicts the fare of a cab ride using real-world features like:
- Pickup and drop location (latitude & longitude)
- Distance calculation using Haversine formula
- Time-based features (hour, day, month)
- Passenger count

The model is deployed using a Flask web application with an interactive UI.

---

## ⚙️ Tech Stack

- Python 🐍
- Flask 🌐
- Scikit-learn 🤖
- Pandas & NumPy 📊
- HTML, CSS, JavaScript 🎨
- Leaflet.js (for map integration) 🗺️

---

## 🧠 Machine Learning Model

- Algorithm: Linear Regression
- Feature Engineering:
  - Distance (Haversine Formula)
  - Hour, Day, Month extraction
- Evaluation Metrics:
  - Mean Absolute Error (MAE)
  - R² Score
